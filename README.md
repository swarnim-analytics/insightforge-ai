# InsightForge AI

AI-powered analytics dashboard for business intelligence, KPI generation, anomaly detection, and local AI insights using Ollama.

---

## Features

- CSV/XLSX uploads
- Automated KPI dashboards
- Interactive analytics visualizations
- AI-generated business insights
- Local LLM integration
- Dockerized deployment

---

## Tech Stack

### Frontend
- Streamlit

### Analytics
- Pandas
- NumPy
- Plotly

### AI
- Ollama

### Infrastructure
- Docker
- Ubuntu Server
- Docker Compose

## Project Goals

This project is designed as a production-quality AI analytics platform showcasing:

- Self-Hosted AI Infrastructure
- Business Intelligence Systems
- Dockerized Python Applications
